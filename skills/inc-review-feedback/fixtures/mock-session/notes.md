# Reviewer session notes — Recipe Box dashboard preview

Reviewer: Sam (mock). Product: "Recipe Box", a meal-planning web app. Surface: the dashboard preview build.

Okay, walking through the dashboard.

First thing - the Save and Cancel buttons are flipped from every other form in the app. Save should be on the right, Cancel on the left. Please swap them.

The page header feels taller than it used to be. I think it was smaller last month? If it actually changed, revert it back - but I might be misremembering.

The two funnel icons in the filter bar confuse me. I honestly can't tell which one filters by cuisine and which by time. I don't know exactly what would fix it - maybe labels under them, maybe merging them into one menu. Something.

The whole recipe grid feels... off? Cramped maybe? I can't put my finger on what. It's the overall feel, not one thing. This is the main surface of the app so I'd want it to feel right.

Why is the Export button always visible in the toolbar? I'd hide it behind the overflow menu - feels like clutter for something people rarely use.

At first I couldn't find the settings page and was going to flag it - oh wait, found it, it's under the avatar menu. Never mind, that's fine.

The placeholder chef-hat avatars should be replaced with our real brand icon set. I know design hasn't delivered those yet - once Priya sends the icon pack this is a quick swap.

At some point we'll want a proper mobile layout for this dashboard. Not for this round, just noting it so it's on the radar - it's why I keep mentioning the grid width stuff.

The date format on recipe cards shows "2026-07-12" - reads a bit robotic, I'd expect "Jul 12". Honestly not a huge deal, if it's annoying to change don't bother.

Also, about this annotation tool itself - the pin marker kept jumping around when I scrolled the page. Made it hard to leave comments. Not about the product, but somebody should know.

One thing I couldn't check: the empty state. The demo account always has recipes in it, so I never saw what a brand-new user sees. No idea if it's built.

That's everything. Overall it's close - the button swap and the filter icons are the ones I'd fix first.
